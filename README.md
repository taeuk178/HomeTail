# HomeTail

## Commit Convention
- feat: 기능 추가
- fix: 버그 수정
- docs: 문서 수정, 변경 ( 주석, MARK, .md )
- refactor: 코드 리팩토링 
- test: 테스트 코드 작성, 수정
- chore: 패키지 매니저 수정 ( cocoapods )

## Code Convention
[Swift-Style-Guide](https://github.com/StyleShare/swift-style-guide)
